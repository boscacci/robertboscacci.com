Blog Post File Structure:
Use: content/blog/YEAR/filename.md
Avoid: _index.md files for posts.

Required frontmatter:

  • title
  • date (ISO format)
  • slug
  • summary
  • draft: false

Recommended:

  • description (SEO, 150-160 chars)
  • keywords
  • lastmod
